блочные элементы всегда занимают всю доступную ширину своего родителя, всегда идут на новой строке после предыдущего элемента, по умолчанию имеют css-свойство { display: block; }

      

      

      

      

      

      


строчные элементы ужимаются под ширину своего контента, идут в одну строку, при нехватке места переносятся на новую строку,  по умолчанию имеют css-свойство { display: inline; }
